# Generated Swift Protobuf Ownership

This directory contains checked-in Swift protobuf output generated from the Android SDK `.proto` schema source of truth in `sources/Android/android-communications/library/src/sdk/proto`.

Do not hand-edit `*.pb.swift` files. Regenerate them with `scripts/generate_swift_protobuf.sh`, which pins `protoc` through the Android Gradle version catalog and verifies the `protoc-gen-swift` version against `Package.swift`.
